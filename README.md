# TcpUdpTool

一款轻量、直观的 TCP/UDP 测试与调试工具，适合在开发阶段快速验证网络通信、消息格式与编码效果。

![ScreenShot](https://user-images.githubusercontent.com/5458667/35699691-16be5620-0791-11e8-91e9-e54df849fa50.png)

## 功能亮点

- TCP 客户端：连接服务器，收发数据、历史记录展示
- TCP 服务端：单连接监听与双向通信
- UDP：收发数据包、支持广播
- UDP 组播：ASM/SSM 模式，支持指定网卡监听
- 发送增强：支持文本/Hex/文件发送，自动发送（毫秒级间隔）
- 转义序列：文本模式下可插入常用转义与 Unicode

## 自动发送（TCP Client）

在 TCP Client 的发送区勾选“自动发送”，并设置间隔（毫秒），即可按固定周期发送当前消息。

生效条件：
- 已建立连接
- 消息内容非空
- 间隔为大于 0 的整数

## 适用场景

- 协议联调与现场排障
- 设备/网关/服务器通信验证
- 抓包前的快速验证与压测
